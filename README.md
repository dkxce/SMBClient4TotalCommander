### Total Commander (Windows) File Plugin (WFX) for SMB2.0/SMB3.0/SMB3.1 Server ###

Main Features:     
==============     
- Allow to change standard port (not 445)
- Supports [SMBLibrary 1.5.1 SVC Server by dkxce](https://github.com/dkxce/SMBLibrary)
